1. 관리자 로그인

관리자 계정으로 로그인하여 쇼핑몰을 관리합니다.

1-1. 접속 방법

1
쇼핑몰 메인 페이지 접속 http://localhost:5173으로 접속합니다.
2
관리자 계정으로 로그인 관리자 이메일과 비밀번호를 입력합니다.
admin@example.com
••••••••
3
관리자 메뉴 접근 로그인 후 상단 메뉴에 "관리자" 메뉴가 표시됩니다. 클릭하면 관리자 페이지로 이동합니다.
🔐 보안 안내: 관리자 계정은 강력한 비밀번호를 사용하고, 주기적으로 변경해주세요.

3. 회원 관리

모든 회원을 조회하고 관리할 수 있습니다.

3-1. 회원 목록 조회

📸 화면 예시: 회원 관리

회원 관리

ID 이름 이메일 유형 가입일 상태 관리

2 김판매 seller@example.com 판매자 2024-12-05 승인대기

3-2. 판매자 승인

판매자로 가입한 회원은 관리자 승인 후 판매 활동이 가능합니다.

1
회원 유형을 "판매자"로 필터링 승인 대기 중인 판매자만 확인할 수 있습니다.
2
사업자 정보 확인 상호명, 사업자등록번호를 확인합니다.
3
"승인" 또는 "거절" 클릭 적합한 판매자는 승인, 부적합하면 거절합니다.
⚠️ 확인 사항: 사업자등록번호가 실제로 존재하는지 국세청 홈택스에서 확인하세요.

4. 상품 관리

판매자가 등록한 상품을 검토하고 승인합니다.

4-1. 상품 승인 대기 목록

📸 화면 예시: 상품 관리

상품 관리

이미지 상품명 판매자 카테고리 가격 등록일 상태 관리
📦
 프리미엄 칫솔 세트 행복구강 칫솔 15,000원 2024-12-14 승인대기

4-2. 상품 승인 기준

확인 항목 기준
상품 이미지 실제 상품 사진인지, 타사 이미지 무단 사용 여부
상품명 명확하고 적절한 이름인지
상품 설명 허위/과대 광고가 없는지
가격 시장가 대비 적정한지
카테고리 올바른 카테고리에 등록되었는지

4-3. 상품 반려 시

상품을 반려할 때는 반드시 사유를 입력합니다. 판매자가 수정 후 재등록할 수 있도록 구체적인 사유를 작성해주세요.

5. 주문 관리

5-1. 주문 목록

5-2. 주문 상세 확인

주문 상세에서 다음 정보를 확인할 수 있습니다:

  • 주문자 정보 (이름, 연락처, 배송지)
  • 주문 상품 목록
  • 결제 정보
  • 배송 상태 및 운송장 번호

5-3. 주문 취소 처리

구매자 요청 시 주문을 취소하고 환불 처리할 수 있습니다.

⚠️ 주의: 이미 배송 중인 주문은 취소가 불가합니다. 반품 절차를 안내하세요.

6. 정산 관리

판매자에게 지급할 정산을 생성하고 처리합니다.

6-1. 정산 주기

항목 내용
정산 주기 월 1회
정산 마감 매월 말일
정산 지급일 익월 15일
수수료 10%

6-2. 정산 생성

매월 말일에 해당 월의 정산을 생성합니다.

1
정산 관리 → "정산 생성" 버튼 클릭 정산 대상 월을 선택합니다.
2
정산 내역 확인 판매자별 매출, 수수료, 환불, 정산금액을 확인합니다.
3
정산 생성 완료 확인 후 정산을 생성하면 판매자에게 알림이 발송됩니다.

6-3. 정산 지급 처리

📸 화면 예시: 정산 목록

정산 관리

정산번호 정산월 판매자 매출 수수료 정산금액 상태 관리
STL-202412-001 2024년 12월 행복구강 950,000원 95,000원 855,000원 대기
STL-202411-001 2024년 11월 행복구강 780,000원 78,000원 702,000원 완료
1
대기 상태 정산 확인 정산 예정일(익월 15일)에 맞춰 정산을 처리합니다.
2
판매자 계좌로 송금 판매자가 등록한 정산 계좌로 정산금을 송금합니다.
3
"지급완료" 버튼 클릭 송금 완료 후 시스템에서 지급완료 처리합니다.
💡 TIP: 정산 상세에서 해당 정산에 포함된 모든 주문 내역을 CSV로 다운로드할 수 있습니다.
