1. 회원가입

쇼핑몰을 이용하려면 먼저 회원가입이 필요합니다.

회원가입 방법

1
홈페이지 접속 웹브라우저에서 http://localhost:5173 주소로 접속합니다.
2
회원가입 버튼 클릭 화면 상단 오른쪽의 "회원가입" 버튼을 클릭합니다.

📸 화면 예시: 상단 메뉴

🦷 오랄케어 칫솔 치약 치실 회원가입 로그인
3
회원정보 입력 아래 정보를 입력합니다.
홍길동
hong@example.com
••••••••
••••••••
010-1234-5678
일반 구매자    판매자
💡 TIP: 비밀번호는 8자 이상으로 설정해주세요. 이메일은 로그인 시 사용됩니다.
4
가입 완료 "회원가입이 완료되었습니다" 메시지가 나타나면 가입이 완료된 것입니다. 이제 로그인할 수 있습니다.

2. 로그인

회원가입 후 로그인하여 쇼핑몰의 모든 기능을 이용할 수 있습니다.

1
로그인 페이지 이동 화면 상단의 "로그인" 버튼을 클릭합니다.
2
로그인 정보 입력 가입 시 입력한 이메일과 비밀번호를 입력합니다.
hong@example.com
••••••••
⚠️ 주의: 비밀번호를 5회 이상 틀리면 계정이 일시적으로 잠길 수 있습니다.

3. 상품 둘러보기

다양한 방법으로 원하는 상품을 찾을 수 있습니다.

3-1. 카테고리로 찾기

상단 메뉴에서 카테고리를 클릭하면 해당 카테고리의 상품 목록을 볼 수 있습니다.

🪥
칫솔

일반/전동 칫솔

🧴
치약

일반/미백/시린이용

🧵
치실

치실/치간칫솔

💧
가글

구강청결제

3-2. 검색으로 찾기

상단 검색창에 상품명을 입력하여 검색할 수 있습니다.

📸 화면 예시: 검색창

칫솔

3-3. 상품 상세 보기

상품 목록에서 원하는 상품을 클릭하면 상세 정보를 볼 수 있습니다.

📸 화면 예시: 상품 상세 페이지

4. 장바구니

구매할 상품을 장바구니에 담아 한 번에 주문할 수 있습니다.

4-1. 장바구니에 담기

1
상품 상세 페이지에서 "장바구니 담기" 클릭 원하는 수량을 선택한 후 장바구니 담기 버튼을 클릭합니다.

4-2. 장바구니 확인

화면 상단의 장바구니 아이콘을 클릭하면 담긴 상품을 확인할 수 있습니다.

📸 화면 예시: 장바구니

🛒 장바구니

상품 가격 수량 합계 삭제
프리미엄 칫솔 세트 15,000원 2 30,000원
미백 치약 100g 8,000원 1 8,000원

총 결제금액: 38,000원

💡 TIP: 장바구니의 상품은 로그아웃해도 유지됩니다. 천천히 고민하고 구매하세요!

5. 주문하기 / 결제하기

장바구니에서 "주문하기"를 클릭하면 결제 페이지로 이동합니다.

5-1. 배송 정보 입력

홍길동
010-1234-5678
서울시 강남구 테헤란로 123
부재시 경비실에 맡겨주세요

5-2. 결제 수단 선택

💳
카드 결제
🏦
계좌이체
📱
간편결제

5-3. 결제 완료

1
"결제하기" 버튼 클릭 모든 정보를 확인한 후 결제하기 버튼을 클릭합니다.
2
결제 진행 선택한 결제 수단에 따라 결제를 진행합니다.
3
주문 완료 결제가 완료되면 주문 완료 페이지가 나타나고, 주문번호가 발급됩니다.

📸 화면 예시: 주문 완료

✅

주문이 완료되었습니다!

주문번호: ORD-20241215-001234

6. 마이페이지

주문 내역 확인, 배송 조회, 회원 정보 수정 등을 할 수 있습니다.

6-1. 마이페이지 접속

로그인 후 화면 상단의 프로필 아이콘 또는 "마이페이지"를 클릭합니다.

6-2. 주문 내역 확인

📸 화면 예시: 주문 내역

📋 주문 내역

주문번호 주문일 상품 결제금액 상태
ORD-20241215-001234 2024-12-15 프리미엄 칫솔 세트 외 1건 38,000원 배송중
ORD-20241210-001122 2024-12-10 구강청결제 500ml 12,000원 배송완료

주문 상태 안내

상태 설명
결제완료 결제가 완료되었으며, 판매자가 상품을 준비 중입니다.
배송중 상품이 발송되어 배송 중입니다. 운송장 번호로 조회 가능합니다.
배송완료 상품이 배송 완료되었습니다.

6-3. 배송 조회

주문 내역에서 "배송조회" 버튼을 클릭하면 택배사 배송 조회 페이지로 이동합니다.

7. 반품 / 교환 신청

상품에 문제가 있거나 변심인 경우 반품 또는 교환을 신청할 수 있습니다.

7-1. 반품/교환 가능 조건

구분 가능 기간 조건
단순 변심 수령 후 7일 이내 상품이 미개봉, 미사용 상태여야 함
상품 불량/오배송 수령 후 30일 이내 사진 또는 영상 증빙 필요

7-2. 반품/교환 신청 방법

1
마이페이지 → 주문 내역 이동 반품/교환할 주문을 찾습니다.
2
"반품/교환 신청" 버튼 클릭 해당 주문의 반품/교환 신청 버튼을 클릭합니다.
3
신청 정보 입력 반품/교환 사유와 상세 내용을 입력합니다.
반품 (환불)    교환 (동일상품)

7-3. 반품/교환 진행 상태

📸 반품/교환 진행 흐름

✓
신청완료
2
판매자 확인
3
상품 반송
4
처리완료
⚠️ 주의: 반품 시 배송비는 단순 변심의 경우 구매자 부담, 상품 불량/오배송의 경우 판매자 부담입니다.
