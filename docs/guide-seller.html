1. 판매자 회원가입

상품을 판매하려면 판매자 계정으로 가입해야 합니다.

1-1. 회원가입 시 "판매자" 선택

1
회원가입 페이지 접속 쇼핑몰 메인 페이지에서 "회원가입" 버튼을 클릭합니다.
2
회원 유형에서 "판매자" 선택 회원 유형 선택 시 반드시 "판매자"를 선택합니다.

1-2. 관리자 승인 대기

판매자 계정은 관리자 승인 후 이용이 가능합니다. 승인까지 1~2 영업일이 소요될 수 있습니다.

3. 상품 등록

판매할 상품을 등록하는 방법입니다.

3-1. 상품 등록 페이지 이동

1
판매자 센터 → 상품 관리 → 상품 등록 좌측 메뉴에서 "상품 관리"를 클릭한 후 "상품 등록" 버튼을 클릭합니다.

3-2. 상품 정보 입력

프리미엄 대나무 칫솔 4개입
15,000
100

이미지를 드래그하거나 클릭하여 업로드

권장: 800x800px, 최대 5MB

3-3. 상품 승인 대기

등록된 상품은 관리자 승인 후 쇼핑몰에 노출됩니다.

상태 설명
승인대기 관리자가 상품을 검토 중입니다.
승인완료 상품이 승인되어 쇼핑몰에 노출됩니다.
반려 상품이 반려되었습니다. 반려 사유를 확인하고 수정 후 재등록하세요.
⚠️ 주의: 상품 이미지는 직접 촬영한 사진을 사용해주세요. 타사 이미지 무단 사용 시 승인이 거부될 수 있습니다.

4-1. 주문 목록 확인

5. 배송 처리

"결제완료" 상태인 주문은 상품을 발송하고 운송장 번호를 입력해야 합니다.

5-1. 배송 처리 방법

1
주문 목록에서 "배송처리" 버튼 클릭 결제완료 상태의 주문에서 배송처리 버튼을 클릭합니다.
2
운송장 정보 입력 택배사와 운송장 번호를 입력합니다.

🚚 배송 정보 입력

123456789012
3
배송 처리 완료 운송장 입력이 완료되면 주문 상태가 "배송중"으로 변경되고, 구매자에게 배송 알림이 발송됩니다.
💡 TIP: 운송장 번호는 정확히 입력해주세요. 구매자가 배송 조회를 할 때 사용됩니다.

6. 정산 확인

판매 대금의 정산을 확인하는 방법입니다.

6-1. 정산 주기

항목 내용
정산 주기 월 1회
정산 마감일 매월 말일
정산 지급일 익월 15일
판매 수수료 10%

6-2. 정산 계산 방식

정산금액 = 총 매출액 - 환불금액 - 수수료(10%)

예시: 매출 100만원 - 환불 5만원 - 수수료 9.5만원 = 85.5만원

6-3. 정산 내역 확인

판매자 센터 → 정산 관리에서 정산 내역을 확인할 수 있습니다.

📸 화면 예시: 정산 내역

6-4. 정산 계좌 설정

정산 관리 → 계좌 설정에서 정산받을 계좌를 등록합니다.

⚠️ 주의: 정산 계좌는 사업자 명의의 계좌로 등록해주세요. 개인 계좌로는 정산이 불가할 수 있습니다.

7. 반품/교환 처리

구매자가 신청한 반품/교환 요청을 처리하는 방법입니다.

7-1. 반품/교환 요청 확인

판매자 센터 → 반품/교환에서 요청 목록을 확인합니다.

📸 화면 예시: 반품/교환 목록

유형 주문번호 상품명 사유 신청일 상태 처리
반품 ORD-20241210-001122 프리미엄 칫솔 단순변심 2024-12-12 대기

7-2. 반품/교환 처리 절차

1
요청 검토 반품/교환 사유와 조건을 확인합니다.
2
승인 또는 거절 조건에 맞으면 "승인", 조건에 맞지 않으면 "거절"을 선택합니다.
3
상품 수거 (승인 시) 구매자가 상품을 반송하면 상품 상태를 확인합니다.
4
처리 완료 반품은 환불 처리, 교환은 새 상품 발송으로 완료합니다.

7-3. 반품 배송비 안내

사유 배송비 부담
단순 변심 구매자 부담
상품 불량 / 오배송 판매자 부담
💡 TIP: 반품 거절 시에는 명확한 사유를 입력해주세요. 분쟁 방지에 도움이 됩니다.
